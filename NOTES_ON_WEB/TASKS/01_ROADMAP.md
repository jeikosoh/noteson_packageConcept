# 01_ROADMAP

이 문서는 "NOTES ON NYC" 웹사이트 구축 프로젝트의 단계별 마일스톤과 작업 추진 일정을 관리하는 로드맵 파일입니다.

## 📅 프로젝트 마일스톤

```mermaid
gantt
    title NOTES ON NYC 구축 일정
    dateFormat  YYYY-MM-DD
    section 기획 및 설계
    옵시디언 노트 세분화      :active, des1, 2026-06-22, 1d
    Figma 연동 워크플로우 설계  : des2, after des1, 1d
    section 랜딩 페이지 개발
    UI 디자인 분석 적용 HTML : des3, after des2, 2d
    인터랙션/애니메이션 보강    : des4, after des3, 2d
    section 콘텐츠 & 배포
    Short Film 페이지 구성    : des5, after des4, 2d
    Admin 업로드 연동        : des6, after des5, 2d
```

## 🛠️ 주요 도구 & 통합 계획
1.  **디자인 툴**: Figma
2.  **코딩 환경**: Vanilla HTML, Vanilla CSS, JS (Micro Interaction)
3.  **동기화 워크플로우**: Anima Plugin 또는 Builder.io를 활용하여 Figma의 시각적 요소와 HTML 코드베이스 간 상호 동기화(양방향).
