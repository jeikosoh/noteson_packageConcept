# 03_WEB_SPECIFICATION

이 문서는 "NOTES ON NYC" 웹사이트의 페이지별 기능 요구사항과 구조적 설계 스펙을 다룹니다.

## 🖥️ 페이지 구성 및 요구사항

### 1. Landing Page (메인 페이지)
*   **Header**: 캡슐 모양(알약형) 내비게이션 바. 브랜드 로고(NOTES ON NYC) 및 링크(Short Films, About, Contact).
*   **Hero Section**: NYC의 현대적 감성을 보여줄 수 있는 시각적 배너 영역.
*   **Feature / Grid Section**: 기계 장치의 다이얼이나 모듈 모양을 시각화한 카드 목록 형태의 디자인 레이아웃.

### 2. Content Page (Short Films / Videos)
*   **비디오 그리드**: 비디오 재생 및 메타데이터(카테고리, 제목, 날짜, 태그)가 테크니컬한 폰트로 표시되는 갤러리.
*   **Admin 기능**: 비디오를 업로드하고 정보를 수정할 수 있는 최소한의 구조 제공 (추후 JSON 파일 관리 또는 CMS 연동으로 세분화).

### 3. About Page
*   **소개**: NOTES ON NYC 브랜드의 비전과 멤버/제작자 소개.
*   **Contact Form**: 피드백 또는 비즈니스 문의를 보낼 수 있는 인터랙티브한 폼.
